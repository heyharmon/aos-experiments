# Actions

| Action | Consequence |
|---|---|
| search | reversible |
| read | reversible |
| file a note | reversible |
| write a draft | reversible |
| prioritize | reversible |
| brief | reversible |
| send email | consequential -> escalate |
| pay or wire money | consequential -> escalate |
| delete | consequential -> escalate |
| book external commitment | consequential -> escalate |

Reversible work: do it directly. Consequential work: escalate by writing a
file to `runtime/queue/approvals/`.
